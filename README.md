Mange Book API – Sua biblioteca em modo desenho animado!

Imagina uma biblioteca mágica, onde cada livro tem vida própria e um guardião digital organiza tudo nos bastidores… Essa é a Mange Book API!
Aqui você cadastra, lista, busca e remove livros como se estivesse controlando um mundinho cartoon de histórias.

Sinopse do Projeto

A Mange Book API é uma API REST feita com Node.js, Express e MongoDB que funciona como o “bibliotecário oficial” do seu sistema.
Ela cuida de um catálogo de livros, permitindo:

Cadastrar livros com título, autor, ano de publicação e ISBN

Listar todos os livros disponíveis na “estante virtual”

Buscar livros pelo título (até por parte do nome, tipo “Dom” em “Dom Casmurro”)

Remover livros pelo ISBN, como se estivesse tirando um livrinho da prateleira

Tudo isso com um toque de organização e um clima de desenho animado nos bastidores.

 Tecnologias usadas

 Node.js

 Express

 MongoDB + Mongoose

 Dotenv

 Thunder Client / Insomnia para testar as rotas

 Objetivo

O objetivo do projeto é praticar conceitos de CRUD, API REST, integração com banco de dados NoSQL e uso de ferramentas de teste de requisições, tudo em um formato simples, direto e divertido – perfeito para trabalhos acadêmicos, estudos e primeiros passos no backend.
